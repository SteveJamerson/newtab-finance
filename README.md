# Finance

Finance is an app to test the knowledge.

## Quick Start

### Install

```bash
npm install
```

_or_

```bash
yarn install
```

### Update environment.js
```bash
export const environment = {
   PATH: "https://api.airtable.com/v0",
   AUTH: "Bearer YOUR_API_KEY",
   KEY: "YOUR_USER_KEY",
   USER: "YOUR_RESPONSIBLE_CODE",
};
```

### Run

```bash
npm run start
```

_or_

```bash
yarn start
```
